Bluetooth Troubleshooting

- Make sure you are using the chrome browser (it supports web bluetooth)
- Make sure bluetooth is turned on on your device
- Pair to the sensor before connecting to it through your browser
- Paste "chrome://flags" into your browser url, search for "Experimental Web Platform features" and make sure it is enabled
